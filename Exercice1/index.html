Résumé de la séance

Aujourd'hui, nous avons eu une séance de cours sur le Développement Web.
Nous avons appris les notions suivantes :

  1. Bref Historique sur le web et sur son fonctionnement
  2. Introduction au HTML
    • Structure de base d'un document HTML
    • Éléments de base : titres ( h1, h2, ... ), paragraphes ( p ), liens (utilisation de "a" et de l'attribut "href" )
    • Faire des listes
      1. Listes non ordonnées ( ul )
      2. Listes ordonnées ( ol)
      3. Listes de définitions ( dl )
    • Mettre en gras ( strong ), Mettre en italique ( em ), Mettre en evidence ( mark )
